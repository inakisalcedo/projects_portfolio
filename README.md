# Data Analysis Portfolio 📈

Welcome to my project portfolio. I'm focused on building data-driven tools for market research, asset valuation, and macroeconomic analysis. This repository showcases my ability to translate financial theories into functional Python code.

---

## 📂 Featured Projects

### ⚖️ Valuation & Fundamental Analysis
* **[Advanced DCF Valuation](./advanced_discounted_valuation.py)**: A multi-stage Discounted Cash Flow (DCF) model that calculates intrinsic value by forecasting free cash flows through dynamic growth phases.
* **[Relative Valuation Scorer](./relative_valuation_scorer.py)**: An interactive tool designed to assess if a stock is undervalued or overvalued using margin-adjusted PEG and PSG ratios.

### 📊 Market Dynamics & Momentum
* **[Weekly RSI Study](./weekly_rsi.py)**: A technical research script that utilizes weekly RSI combined with Moving Averages to identify long-term trend exhaustion and momentum shifts.
